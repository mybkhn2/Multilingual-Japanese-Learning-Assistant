Privacy Policy

Effective Date: April 20, 2025

The Japanese Learning Assistant Extension is designed to help users translate and learn Japanese through context-aware translation, Kanji reading, and grammar explanation. We take your privacy seriously.

1. Data Collection

This extension does not collect, store, or share any personal information from users. The only data used is the text you select on a webpage, which is temporarily sent to the OpenAI API for the purpose of generating a translation and explanation.

2. Use of OpenAI API

When you select Japanese text and trigger the extension, your selected sentence is sent securely to the OpenAI API. The response includes translation, Kanji readings, grammar notes, and pronunciation. No identifying information is attached or transmitted.

3. Local Storage

The extension uses Chromeâ€™s storage.sync to remember your selected language. This is stored locally in your browser and is not transmitted or shared.

4. No Third-Party Access

We do not sell, trade, or transfer your data to any third parties. All operations happen locally or securely via OpenAI.

5. Website Content Access

This extension temporarily accesses only the text that users manually select on any website in order to provide translations, Kanji readings, and grammar explanations. It does not read or access other elements such as images, links, videos, or the full page content. No data is stored or collected. The selected text is used only for the purpose of sending to the OpenAI API and is discarded immediately after a response is received.

6. Changes to Policy

If there are any changes to this privacy policy, they will be published directly here in this document with an updated date.

7. Contact

If you have any questions or concerns regarding this policy, please contact us via the Chrome Web Store support page.

Thank you for using our extension!
