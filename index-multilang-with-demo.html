🌐 Japanese Learning Assistant - Multilingual Extension

This Chrome extension allows users to select Japanese text on any website and instantly translate it into one of several supported languages.

✨ Key Features

This extension doesn't just translate – it also explains Kanji readings and basic grammar patterns from the selected sentence. Great for Japanese learners and native speakers alike!

  • Select Japanese text → tooltip appears → translate
  • Supports 6 languages: Vietnamese, English, Spanish, Sinhala, Korean, French
  • Language switcher with ❌ close button

How It Works

  1. Install the extension
  2. Select Japanese text → click tooltip
  3. Translation appears in a popup

🌐 Trợ Lý Học Tiếng Nhật - Hỗ Trợ Nhiều Ngôn Ngữ

Tiện ích Chrome này giúp bạn bôi đen câu tiếng Nhật và dịch sang ngôn ngữ bạn chọn (tiếng Việt, Anh, Hàn, Pháp, v.v.).

✨ Tính năng nổi bật

Không chỉ dịch, tiện ích còn phân tích cách đọc các từ Kanji và giải thích ngữ pháp có trong câu được chọn. Cũng hỗ trợ học tiếng Anh cho người Nhật.

  • Bôi đen văn bản → hiện tooltip → click để dịch
  • Hỗ trợ 6 ngôn ngữ: Việt, Anh, Tây Ban Nha, Sri Lanka, Hàn, Pháp
  • Chọn ngôn ngữ ở góc dưới, có nút ❌ để ẩn

Cách sử dụng

  1. Cài tiện ích vào trình duyệt
  2. Bôi đen câu tiếng Nhật → click tooltip
  3. Kết quả sẽ hiện ra trong popup nhỏ

🌐 日本語学習アシスタント - 多言語対応拡張機能

このChrome拡張機能は、Webサイト上の日本語の文章を選択して、選択した言語に翻訳できます。

✨ 主な機能

翻訳だけでなく、選択された文に含まれる漢字の読み方や文法の解説も表示されます。日本人の英語学習にも便利です。

  • 日本語を選択 → ツールチップ表示 → 翻訳
  • 対応言語：ベトナム語、英語、スペイン語、シンハラ語、韓国語、フランス語
  • 画面右下に言語切替メニュー（❌で閉じる）

使い方

  1. Chromeに拡張機能をインストール
  2. 日本語の文を選択 → ツールチップをクリック
  3. 数秒後に翻訳結果がポップアップで表示
